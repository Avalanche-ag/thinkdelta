How This Works

01

The Challenge

GitHub Pages is static-only. MiniMax API doesn't support CORS for client-side calls. A live backend proxy would be needed for real-time AI.

02

The Solution

Pre-generated 6 real MiniMax M2.7 responses via Python script. Static JSON serves as a snapshot. Full live-API code architecture is in app.js — swap the JSON fetch for an fetch() to your proxy and it works.

03

Live API Path

Run python3 generate_examples.py --key YOUR_MINIMAX_KEY to regenerate with real responses. Or deploy a Cloudflare Worker proxy and uncomment the live API block in app.js.
